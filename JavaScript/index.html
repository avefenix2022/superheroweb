Hola Mundo! Javascript

En un archivo Script.js se añade el siguiente código:

function holamundo(){
    alert("HOLA MUNDO");
}

function hace referencia a una función que encierra una serie de líneas de código que se van a ejecutar cada vez que se llame a la función holamundo()

alert() muestra una ventana emergente con el texto encerrado dentro de los paréntesis.

Las instrucciones en javascript finalizan todas en ;

Para vincular este archivo Script.js al html se añade la cadena:

<script src='Script.js'></script>

Lo siguiente es asignar a un evento una llamada a esa función, por ejemplo al hacer clic en el siguiente botón Call to action!

<a class="btn btn-primary btn-large" onclick="holamundo()">Call to action!</a>

Call to action!

Se puede insertar llamadas a Javascript desde el código html, por ejemplo a la función:

function mostrarHoy(){
    var hoy = new Date();
    document.write(hoy.toDateString());
}

se la puede llamar desde el html:

<p>La fecha de hoy es: <script>mostrarHoy();</script></p>

Hola Mundo! jQuery

jQuery es una biblioteca de Javascript cuyo lema es:

"escribir menos, hacer más"

Hace que la programación sea divertida, rápida y reconfortante.

Con jQuery, en una sola línea de código se puede conseguir lo mismo que con 100 de Javascript.

Por ejemplo el código de jQuery:

function animacion() {
    $("body").hide().fadeIn(3000);
}

Y en el html añadir en el botón el evento onclick:

onclick="$(document).ready(animacion);
